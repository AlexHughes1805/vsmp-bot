# 2006 VSMP Bot
2000’s VSMP bot has been produced as an aid to a roleplay server set in an early 2000’s era of POWCreation’s Vampires SMP. The events of the original VSMP is canon history to this roleplay.

The bot has connectivity with MongoDB to store user information such as inventory items or adventuring party status.
